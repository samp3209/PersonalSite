Reinforcement Learning for Health Policies

Technologies: Python, Mesa-Agent-Based Modeling

Using Reinforcement Learning to Dynamically Update Health Policies during a Pandemic. This project simulates and optimizes health policy responses in real-time using learned weights from a Double DQN. Updates different health policies every 24 steps in the simulation based on current infection and production rates.

A versatile image manipulation tool supporting both GUI and command-line interfaces. Features include filters, transformations, and batch processing. Digital Assets to Promotional Content

Technologies: BLIP, Stable Diffusion, Python

Capstone project transforming digital assets into artificially generated promotional content using BLIP for labeling and fine-tuned Stable Diffusion models. Using BLIP I demonstrated that one could quickly mass label image assets and use that to train a Stable Diffusion model to produce artificially generated logos and promotional material.
